5.1 双极结型三极管（BJT）

动机：需要电流放大器件来构建放大电路。

BJT：双极结型三极管，有两种载流子（电子和空穴）参与导电。

BJT的结构 重点

结构特点：发射区掺杂浓度最高，基区很薄且掺杂浓度低，集电区面积大。

三种组态：共基极（CB）、共发射极（CE）、共集电极（CC）。

BJT的工作原理 必考

放大条件：发射结正偏，集电结反偏。

电流关系：

$$I_C = \beta I_B$$ $$I_E = (1 + \beta)I_B$$ $$\beta = \frac{I_C}{I_B}$$

$\beta$ 为电流放大系数，通常为几十到几百

易错：BJT是电流控制器件，$I_C = \beta I_B$；MOSFET是电压控制器件，$I_D$ 受 $V_{GS}$ 控制。

BJT的I-V特性曲线 必考

输出特性曲线（共射极）：

区域条件特点
截止区$i_B = 0$$i_C \approx 0$
放大区发射结正偏，集电结反偏$i_C = \beta i_B$
饱和区发射结正偏，集电结正偏$v_{CE} < v_{BE}$

5.2 BJT放大电路

基本共射极放大电路 必考

静态工作点：$I_{BQ}$、$I_{CQ}$、$V_{CEQ}$

$$I_{BQ} = \frac{V_{CC} - V_{BEQ}}{R_b}$$ $$I_{CQ} = \beta I_{BQ}$$ $$V_{CEQ} = V_{CC} - I_{CQ}R_c$$

BJT的小信号模型 必考

H参数模型：$r_{be}$（输入电阻）和 $\beta i_b$（受控电流源）。

输入电阻：

$$r_{be} = r_{bb'} + (1 + \beta)\frac{V_T}{I_{EQ}}$$

$r_{bb'}$ 为基区体电阻，通常约200$\Omega$

共射极放大电路动态指标 必考

$$A_v = -\beta \frac{R_c // R_L}{r_{be}}$$ $$R_i = R_b // r_{be}$$ $$R_o = R_c$$
