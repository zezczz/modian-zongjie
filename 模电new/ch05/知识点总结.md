# 第五章 双极结型三极管及其放大电路

## 5.1 双极结型三极管（BJT）
### 5.1.1 BJT的结构简介
- **结构**：NPN型和PNP型。
- **结构特点**：
  - 发射区掺杂浓度最高。
  - 集电区掺杂浓度最低，面积大。
  - 基区很薄，掺杂浓度低于发射区。

### 5.1.2 BJT的工作原理
- **放大作用条件**：
  - 内部条件：发射区杂质浓度远大于基区杂质浓度，且基区很薄。
  - 外部条件：发射结正向偏置，集电结反向偏置。
- **电流关系**：
  - $I_C = \alpha I_E$
  - $I_C = \beta I_B$
  - $I_E = (1 + \beta) I_B$
- **三种组态**：共基极、共发射极、共集电极。

### 5.1.3 BJT的I-V特性曲线
- **共基极连接**：
  - 输入特性曲线：$i_E = f(v_{BE})|_{v_{CB}=const}$
  - 输出特性曲线：$i_C = f(v_{CB})|_{i_E=const}$
- **共射极连接**：
  - 输入特性曲线：$i_B = f(v_{BE})|_{v_{CE}=const}$
  - 输出特性曲线：$i_C = f(v_{CE})|_{i_B=const}$
- **输出特性曲线的三个区域**：
  - **截止区**：$i_C$ 接近零。
  - **放大区**：$i_C$ 平行于 $v_{CE}$ 轴，$i_C = \beta i_B$。
  - **饱和区**：$i_C$ 明显受 $v_{CE}$ 控制。

### 5.1.4 BJT的主要参数
- **电流放大系数**：$\alpha$、$\beta$。
- **极间反向电流**：$I_{CBO}$、$I_{CEO}$。
- **极限参数**：$I_{CM}$、$P_{CM}$、$V_{(BR)CEO}$。

## 5.2 BJT放大电路
### 5.2.1 基本共射极放大电路
- **静态工作点（Q点）**：$I_{BQ}$、$I_{CQ}$、$V_{CEQ}$。
- **动态工作过程**：输入信号引起 $i_B$、$i_C$、$v_{CE}$ 的变化。

### 5.2.2 BJT放大电路的图解分析
- **静态工作点**：直流负载线与输出特性曲线的交点。
- **动态工作情况**：工作点沿负载线移动。

### 5.2.3 BJT的小信号模型
- **H参数小信号模型**：
  - $r_{be}$：输入电阻。
  - $\beta$：电流放大系数。
  - $r_{ce}$：输出电阻。
- **简化模型**：忽略 $r_{ce}$。
- **$r_{be}$ 的估算**：$r_{be} = r_{bb'} + (1 + \beta) \frac{V_T}{I_{EQ}}$。

### 5.2.4 共射极放大电路
- **静态工作点计算**：
  - $I_{BQ} = \frac{V_{CC} - V_{BEQ}}{R_b}$
  - $I_{CQ} = \beta I_{BQ}$
  - $V_{CEQ} = V_{CC} - I_{CQ} R_c$
- **动态指标**：
  - **电压增益**：$A_v = -\beta \frac{R_c // R_L}{r_{be}}$
  - **输入电阻**：$R_i = R_b // r_{be}$
  - **输出电阻**：$R_o \approx R_c$

### 5.2.5 共集电极和共基极放大电路
- **共集电极放大电路（射极跟随器）**：
  - 电压增益：$A_v \approx 1$
  - 输入电阻：很高
  - 输出电阻：很低
- **共基极放大电路**：
  - 电压增益：$A_v = \beta \frac{R_c // R_L}{r_{be}}$
  - 输入电阻：很低
  - 输出电阻：≈ $R_c$

## 5.3 FET和BJT及其基本放大电路性能的比较
- **FET**：电压控制电流器件（VCCS），输入电阻高。
- **BJT**：电流控制电流器件（CCCS），输入电阻较低。

## 5.4 多级放大电路
### 5.4.2 共集−共集放大电路
- **复合管**：提高电流放大系数。
- **复合管特性**：$r_{be} = r_{be1} + (1 + \beta_1) r_{be2}$。
