1.1 信号

信号是信息的载体或表达形式——温度传感器输出的电压就是温度信息的载体。

电信号的电路表示：信号在电路中以电压或电流的形式传输和处理。

1.2 信号的频谱

动机：时域波形难以分析电路响应，需要转换到频域。

正弦信号：最基本的信号形式，任何周期信号都可分解为正弦信号的叠加。

方波信号的傅里叶级数展开：满足狄里赫利条件的周期信号可展开为傅里叶级数。

$$v(t) = \frac{4V_m}{\pi} \sum_{n=1,3,5,...}^{\infty} \frac{1}{n} \sin(n\omega_0 t)$$

这个公式说明方波由奇次谐波分量叠加而成。

非周期信号的傅里叶变换：将时域信号变换为频域表示。

$$V(j\omega) = \int_{-\infty}^{\infty} v(t) e^{-j\omega t} dt$$
易错：实际电路的处理能力是有限的，只能处理一定带宽内的信号。

1.4 放大电路模型

动机：需要将微弱信号放大到足以驱动负载的水平。

放大电路是一个双口网络：输入端口接收信号，输出端口驱动负载。

增益形式：

增益类型定义分贝表示
电压增益$A_v = v_o/v_i$$20\lg|A_v|$ dB
电流增益$A_i = i_o/i_i$$20\lg|A_i|$ dB
功率增益$A_p = P_o/P_i$$10\lg A_p$ dB

电压放大模型 必考

输入回路：信号源内阻 $R_s$ 与放大电路输入电阻 $R_i$ 分压。

$$v_i = \frac{R_i}{R_s + R_i} v_s$$

输出回路：受控电压源 $A_{vo}v_i$ 与输出电阻 $R_o$、负载 $R_L$ 串联。

$$v_o = \frac{R_L}{R_o + R_L} A_{vo} v_i$$

要想减小负载对增益的影响，需要 $R_o \ll R_L$；要想减小信号源内阻的影响，需要 $R_i \gg R_s$。

1.5 放大电路的主要性能指标

输入电阻 重点

$R_i = v_i/i_i$：反映放大电路对信号源的负载效应。$R_i$ 越大，对信号源的负载越小。

输出电阻 重点

$R_o$：反映放大电路带负载的能力。$R_o$ 越小，带负载能力越强。

易错：输入、输出电阻是交流电阻，计算时需将独立源置零。

频率响应 必考

频率响应：输出随输入信号频率连续变化的稳态响应。

幅度失真：放大电路对不同频率分量的放大倍数不同。

相位失真：放大电路对不同频率分量产生的时延不同。

频率失真（线性失真）：幅度失真和相位失真的统称。

非线性失真 高频

非线性失真：由元器件非线性特性引起的失真。

$$\gamma = \frac{\sqrt{V_{o2}^2 + V_{o3}^2 + ...}}{V_{o1}} \times 100\%$$

$\gamma$ 为非线性失真系数，$V_{o1}$ 是基波分量有效值，$V_{ok}$ 是高次谐波分量有效值。
