# 第十一章 直流电源电路

## 11.1 小功率整流滤波电路
- **整流电路**：利用二极管的单向导电性，将交流电转换为脉动直流电。
- **滤波电路**：滤除脉动直流电中的纹波，输出较平滑的直流电。
- **电容滤波电路**：
  - 二极管的导电角 $\theta < \pi$，流过二极管的瞬时电流很大。
  - 负载直流平均电压 $V_L$ 升高，$\tau_d = R_L C$ 越大，$V_L$ 越高。
  - 当 $\tau_d \geq (3 \sim 5) T/2$ 时，$V_L = (1.1 \sim 1.2) V_2$
- **电感滤波电路**：
  - 整流二极管的导电角增大，没有峰值电流，输出特性变得平滑。
  - 缺点：体积大，笨重，容易引起电磁干扰。

## 11.2 线性稳压电路
### 11.2.1 线性串联反馈式稳压电路
- **输出电压固定的稳压电路**：
  - 电压串联负反馈，稳定输出电压。
  - 对基准电压 $V_{REF}$ 的稳定性要求很高。
- **输出电压可调的稳压电路**：
  - 输出电压 $V_O = (1 + \frac{R_2}{R_1}) V_{REF}$
- **低压差稳压电路（LDO）**：
  - 使用PMOS管，输入输出压差可以很小。
  - 效率更高，低电压情况下更具优势。

### 11.2.2 线性集成稳压器及其应用举例
- **集成稳压器简介**：
  - 输出电压固定的线性三端集成稳压器：78XX（正电压）、79XX（负电压）。
  - 输出电压可调的线性集成三端稳压器：LM317（正电压）、LM337（负电压）。
- **应用举例**：
  - 典型应用电路，保护二极管。
  - TPS79901等低压差稳压器。

## 11.3 开关稳压电路
- **特点**：
  - 效率高：调整管工作于开关状态，管耗很小，效率可达75%～95%。
  - 体积小，重量轻：省去了笨重的工频变压器和散热装置。
  - 稳压范围宽：效率与输入电压的大小基本无关。
  - 调整管的控制比较复杂。
  - 输出电压纹波较大。

### 11.3.1 无变压器的开关稳压电路
- **开关DC/DC变换器基本电路**：
  - **串联型（降压型或Buck型）**：$V_O = q V_I$，$q = t_{on}/T$ 为占空比。
  - **并联型（升压型或Boost型）**：$V_O > V_I$。
  - **极性反转型**：$V_O$ 极性与 $V_I$ 相反。
- **稳压原理**：脉宽调制（PWM），反馈控制使 $v_F$ 无限趋近于 $V_{REF}$。
- **集成开关稳压器及其应用举例**：
  - LM2596-5.0（固定输出降压型）。
  - LM2576-ADJ（可调降压型）。
  - MAX764（极性反转型）。

### 11.3.2 带变压器的开关稳压电路
- **基本电路及工作原理**：
  - 反激式，通过开关管T1将高压直流变为高频方波电压，再通过高频变压器降压。
- **集成控制器及开关稳压电源**：
  - 包含开关管的集成控制器，高、低压电路隔离。
