10.1 滤波电路的基本概念与分类

动机：需要从复杂信号中提取有用频率成分，抑制无用成分。

滤波器：能使有用频率信号通过，同时抑制或衰减无用频率信号的电子装置。

滤波器分类 必考

类型通带阻带应用
低通（LPF）0~$f_H$$f > f_H$抑制高频干扰
高通（HPF）$f > f_L$0~$f_L$抑制低频干扰
带通（BPF）$f_L$~$f_H$$f < f_L$，$f > f_H$选频
带阻（BRF）$f < f_L$，$f > f_H$$f_L$~$f_H$抑制特定频率

有源滤波器 重点

有源滤波器：由有源器件（运放）和RC网络组成的滤波器。

优点：不用电感，体积小，有一定的电压放大作用。缺点：工作频率受限，不适宜大功率场合。

10.2 一阶有源滤波电路

一阶低通：传递函数 $A(s) = A_0/(1 + s/\omega_c)$

一阶高通：传递函数 $A(s) = A_0 s/\omega_c/(1 + s/\omega_c)$

一阶滤波器的带外衰减速率较慢（-20dB/十倍频）

10.3 高阶有源滤波电路

二阶压控电压源型低通滤波器 必考

传递函数：

$$A(s) = \frac{A_0 \omega_c^2}{s^2 + \frac{\omega_c}{Q}s + \omega_c^2}$$

$A_0$：通带增益，$\omega_c$：特征角频率，$Q$：品质因数

稳定条件：$A_0 < 3$

巴特沃斯滤波器：$Q = 0.707$ 时，通带内幅频响应最平坦。

10.5 正弦波振荡电路的振荡条件

振荡条件 必考

振幅平衡条件：$|AF| = 1$

相位平衡条件：$\phi_A + \phi_F = 2n\pi$

起振条件：$|AF| > 1$

振荡电路组成 重点

  • 放大电路
  • 反馈网络（正反馈）
  • 选频网络
  • 稳幅环节

10.6 RC正弦波振荡电路

RC桥式振荡电路 必考

选频特性：RC串并联网络在 $\omega_0 = 1/(RC)$ 时，反馈系数最大为1/3。

振荡条件：$A_v = 1 + R_f/R_1 \geq 3$

振荡频率：$f_0 = 1/(2\pi RC)$

RC振荡电路用于产生1MHz以下的低频正弦波

稳幅措施 高频

热敏电阻：利用热敏电阻的非线性特性自动调节增益。

场效应管：利用场效应管的压控电阻特性实现稳幅。

10.8 非正弦信号产生电路

电压比较器 必考

单门限比较器：只有一个门限电压，输入超过门限输出跳变。

迟滞比较器：有两个门限电压，抗干扰能力强。

方波产生电路 重点

组成：迟滞比较器 + RC充放电支路 + 稳压管限幅。

振荡周期：$T = 2RC\ln(1 + 2R_1/R_2)$

锯齿波产生电路 高频

由同相输入迟滞比较器和积分电路组成，通过不同的充放电时间常数产生锯齿波。
