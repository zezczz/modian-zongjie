# 第十章 信号处理与信号产生电路

## 10.1 滤波电路的基本概念与分类
- **滤波器**：能使有用频率信号通过，同时抑制或衰减无用频率信号的电子装置。
- **有源滤波器**：由有源器件和相关元件构成的滤波器。
- **滤波电路的传递函数**：
  - 幅频响应：$|A(j\omega)|$
  - 相频响应：$\phi(\omega)$
  - 群时延响应：$\tau(\omega) = -\frac{d\phi}{d\omega}$
- **分类**：
  - **低通（LPF）**
  - **高通（HPF）**
  - **带通（BPF）**
  - **带阻（BRF）**
  - **全通（APF）**

## 10.2 一阶有源滤波电路
- **一阶有源低通滤波电路**：
  - 传递函数：$A(s) = \frac{A_0}{1 + s/\omega_c}$
  - 特征角频率：$\omega_c = 1/RC$
- **一阶有源高通滤波电路**：
  - 传递函数：$A(s) = \frac{A_0 s/\omega_c}{1 + s/\omega_c}$
- **优点**：不用电感、体积小、重量轻、有一定的电压放大和缓冲作用。
- **缺点**：工作频率难以做得很高，不适宜大功率场合。

## 10.3 高阶有源滤波电路
### 10.3.1 有源低通滤波电路
- **二阶压控电压源型**：
  - 传递函数：$A(s) = \frac{A_0 \omega_c^2}{s^2 + \frac{\omega_c}{Q}s + \omega_c^2}$
  - 通带增益 $A_0$、特征角频率 $\omega_c$、等效品质因数 $Q$。
  - 稳定工作条件：$A_0 < 3$。
- **二阶无限增益多路反馈型**：
  - $Q$ 不会出现负值，对增益无限制。
- **高阶滤波器**：通过级联二阶或一阶有源滤波电路构成。
- **巴特沃斯滤波器**：当 $Q=0.707$ 时，通带内幅频响应曲线最大限度平坦。

### 10.3.2 有源高通滤波电路
- 将低通电路中的电容和电阻对换，便成为高通电路。

### 10.3.3 有源带通滤波电路
- 可由低通和高通串联得到。
- 截止角频率：$BW = f_0 / Q$

### 10.3.4 有源带阻滤波电路
- 可由低通和高通并联得到。
- 双T选频网络。

## 10.5 正弦波振荡电路的振荡条件
- **振荡条件**：
  - **振幅平衡条件**：$|AF| = 1$
  - **相位平衡条件**：$\phi_A + \phi_F = 2n\pi$
- **起振条件**：$|AF| > 1$
- **振荡电路基本组成部分**：
  - 放大电路
  - 反馈网络（构成正反馈）
  - 选频网络
  - 稳幅环节

## 10.6 RC正弦波振荡电路
- **电路组成**：RC桥式振荡电路。
- **RC串并联选频网络的选频特性**：
  - 反馈系数 $F = \frac{1}{3 + j(\omega RC - 1/(\omega RC))}$
  - 当 $\omega = \omega_0 = 1/RC$ 时，$F$ 有最大值 $1/3$。
- **振荡电路工作原理**：
  - 满足相位平衡条件：$\phi_A + \phi_F = 0$
  - 满足振幅平衡条件：$A \geq 3$
- **稳幅措施**：
  - 热敏电阻
  - 场效应管（JFET）
  - 二极管
- **RC移相式正弦波振荡电路**。

## 10.8 非正弦信号产生电路
### 10.8.1 电压比较器
- **单门限电压比较器**：
  - 工作于开环状态，虚短不再成立。
  - 当 $v_I > V_{REF}$ 时，$v_O = V_{OH} = +V_{CC}$
  - 当 $v_I < V_{REF}$ 时，$v_O = V_{OL} = -V_{CC}$
- **迟滞比较器**：
  - 门限电压随输出电压变化。
  - 上门限电压、下门限电压、回差电压。
  - 抗干扰能力强。
- **集成电压比较器**：转换速率高，响应时间30～200ns。
- **窗口比较器**：LM339。

### 10.8.2 方波产生电路
- **电路组成**：迟滞比较器 + RC充放电支路 + 稳压管双向限幅。
- **工作原理**：RC充放电过程。
- **振荡周期**：$T = 2RC \ln(1 + \frac{2R_1}{R_2})$
- **占空比调节**。

- 充放电时间常数不同。
