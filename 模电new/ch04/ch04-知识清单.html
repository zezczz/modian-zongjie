4.1 MOS场效应三极管

动机：BJT是电流控制器件，输入电阻低；需要电压控制、高输入电阻的器件。

MOSFET：金属-氧化物-半导体场效应晶体管，电压控制电流器件（VCCS）。

N沟道增强型MOSFET 必考

结构：栅极G、源极S、漏极D、衬底B。栅极绝缘，输入电阻极高。

工作原理：

  • $v_{GS} < V_{TN}$：无导电沟道，$i_D = 0$（截止）
  • $v_{GS} > V_{TN}$：形成导电沟道（反型层），$i_D$ 受 $v_{GS}$ 控制
  • $V_{TN}$：阈值电压（开启电压）

输出特性曲线：

区域条件特点
截止区$v_{GS} < V_{TN}$$i_D = 0$
可变电阻区$v_{DS} < v_{GS} - V_{TN}$$i_D$ 与 $v_{DS}$ 近似线性
恒流区（放大区）$v_{DS} \geq v_{GS} - V_{TN}$$i_D$ 基本不变

恒流区I-V特性：

$$i_D = K_n(v_{GS} - V_{TN})^2(1 + \lambda v_{DS})$$

$K_n$ 为电导常数，$\lambda$ 为沟道长度调制系数

易错：MOSFET是电压控制器件，栅极电流几乎为零；BJT是电流控制器件，基极电流控制集电极电流。

P沟道MOSFET 高频

与N沟道MOSFET工作原理类似，但电压极性和电流方向相反。增强型P沟道MOSFET的 $V_{TP} < 0$。

4.2 MOSFET基本共源极放大电路

电路组成：MOSFET + 偏置电路 + 负载电阻。

静态工作点：$V_{GSQ}$、$I_{DQ}$、$V_{DSQ}$，需验证工作在恒流区。

小信号模型分析法 必考

MOSFET小信号模型：电压控制电流源（VCCS），$g_m v_{gs}$。

跨导：

$$g_m = 2K_n(V_{GSQ} - V_{TN})(1 + \lambda V_{DSQ}) \approx 2K_n(V_{GSQ} - V_{TN})$$

$g_m$ 反映 $v_{GS}$ 对 $i_D$ 的控制能力，单位：S（西门子）或 A/V

共源放大电路动态指标：

$$A_v = -g_m(R_D // R_L)$$ $$R_i = R_{G1} // R_{G2}$$ $$R_o = R_D$$

4.3 图解分析法

直流负载线：$v_{DS} = V_{DD} - i_D R_D$
